# Wordle-Game
#Wordle — это словесная головоломка, в которой игрок должен угадать пятибуквенное слово за ограниченное количество попыток. После каждой попытки система предоставляет обратную связь, которая помогает сузить круг возможных вариантов
[✔]- на своем месте
[≈]- не на своем месте, но присутствует в слове
[✖]- нет в слове
